# AI Daily - 每日 AI 技术日报

AI Daily Digest 是一个每日 AI 技术资讯网站，聚合精选全球 AI 领域的最新动态，支持中英文切换和语音播放功能。

## 技术栈

- **框架**: Next.js (App Router)
- **样式**: Tailwind CSS
- **语言**: TypeScript

## 开发

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看页面。

## 项目结构

```
app/
  layout.tsx   -- 全局布局与 metadata
  page.tsx     -- 首页
  globals.css  -- 全局样式
```
